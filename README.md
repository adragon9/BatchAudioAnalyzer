# Description
This is a utility that analyzes folders of audio files and outputs json data with their tempo's and beat offsets, it was made for the game Dead as Disco to make importing songs easier, though it could be useful elsewhere.

# VENV (Virtual Environment)
- python -m venv /path/to/new/virtual/environment
- source .venv/bin/activate  or .venv\Scripts\activate on Windows
- pip install -r requirements.txt

Your venv path should be where the batch_analysis.py is. In case you need more (https://docs.python.org/3/library/venv.html#creating-virtual-environments)
